# Aegis — Autonomous Denials Management & Appeals Agent (POC)

A demonstration of a 24/7 self-evolving AI agent that triages every payer claim
denial and drafts winning appeals automatically. **All data is fully synthetic**
(generated by `generator/generate.py`) — no real patient information.

## What it shows
- **5,000 synthetic denied claims** + 800-row historical "learning ledger" + 30 clinical-note snippets.
- **Triage engine**: expected recovery = billed × historical overturn prob (payer×CARC) × deadline urgency.
- **Appeal-letter generator**: payer-addressed letters citing the clinical evidence and the strongest historical argument for that payer/denial type.
- **Self-evolution**: win rate climbing over 12 simulated weeks as the ledger updates.
- **Dashboard**: KPI header, filterable work queue, claim → reasoning-chain + full appeal-letter modal, 4 Chart.js charts, live agent activity feed.

## Run locally
```
python generator/generate.py     # regenerate data/ (optional; already committed)
npm start                        # serves ./public on $PORT (default 3000)
```
